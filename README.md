# I2C_OLED-screensaver-DVD-
A screen saver for cheap i2c oled displays , A vintage DVD  one where A DVD logo just keeps bouncing off the corners


this code was flashed to esp32 ( default one) in rust using espidf
if you want you can directly flash this but to build it make sure you first activate the idf environment

and make sure you've got all the dependancies install 
just do as cargo says
